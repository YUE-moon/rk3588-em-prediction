# rk3588-em-prediction

## 项目简介

本作品基于瑞芯微RK3588开发板（ELF2），设计了一套主动式小区域电磁环境实时建模与损耗预测系统。系统通过RK3588主控芯片控制发射信标，接收天线端频谱仪获取 IQ 基带数据，并融合气象、高度、方位等多传感器信息。采集数据经预处理后送入RK3588内置NPU，运行经RKNN工具链部署的神经网络推理模型，完成对当前场景电磁传输损耗的毫秒级预测。

系统采用"PC离线训练 + 板端边缘推理"的协同架构：板端可使用已部署模型进行推理预测，同时支持数据采集与回传，交由PC端进行神经网络训练后优化板端模型，实现模型闭环迭代。

## 技术架构

| 模块 | 说明 |
|------|------|
| **硬件平台** | Rockchip RK3588（ELF2 开发板），6 核 CPU + 6 TOPS NPU |
| **核心算法** | 神经网络回归模型（路径损耗预测） |
| **部署工具链** | RKNN-Toolkit2（PC 端）+ RKNPU2（板端） |
| **数据来源** | 频谱仪 IQ 基带数据 + 气象/高度/方位传感器 |
| **架构模式** | PC 离线训练 → 模型转换 → 板端 NPU 推理 → 数据回传 → 模型迭代 |

## 快速开始

### 环境要求

- **PC端**：Python 3.8+，RKNN-Toolkit2
- **板端**：RK3588 Linux 系统，RKNPU2 运行时

### 模型部署

```bash
# 1. PC 端训练并转换为 RKNN 格式
python 3_核心代码/neural_network/train.py
python 3_核心代码/rknn_model/model_deploy.py

# 2. 将 .rknn 模型推送到板端
scp 4_模型文件/model.rknn root@<板端IP>:/opt/model/

# 3. 板端运行推理
python 3_核心代码/rknn_model/inference_demo.py
```
