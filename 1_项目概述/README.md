# 项目概述

## 项目名称

基于 RK3588 的电磁环境实时建模与边缘 AI 预测系统

## 项目背景

在复杂电磁环境中，准确的电磁传输损耗预测对通信组网、频谱管理和电磁态势感知具有重要意义。传统路径损耗模型（如 Okumura-Hata、COST-231 等）依赖经验公式，难以适应复杂多变的实际场景。本项目利用深度学习方法，结合边缘计算能力，实现小区域电磁环境的实时建模与损耗预测。

## 系统概述

本作品基于瑞芯微 RK3588 开发板（ELF2），设计了一套主动式小区域电磁环境实时建模与损耗预测系统。系统通过 RK3588 主控芯片控制发射信标，接收天线端频谱仪获取 IQ 基带数据，并融合气象、高度、方位等多传感器信息。采集数据经预处理后送入 RK3588 内置 NPU，运行经 RKNN 工具链部署的神经网络推理模型，完成对当前场景电磁传输损耗的毫秒级预测。

## 系统架构

采用 **"PC 离线训练 + 板端边缘推理"** 的协同架构：

1. **数据采集阶段**：板端控制信标发射，频谱仪采集 IQ 数据，传感器同步采集环境数据
2. **边缘推理阶段**：板端 NPU 运行神经网络模型，实时输出传输损耗预测值
3. **模型迭代阶段**：板端收集的数据回传至 PC，PC 端重新训练并优化模型，再部署回板端

## 创新点

- **主动式建模**：通过控制发射信标主动构建已知条件的测试场景，提高数据质量
- **多源数据融合**：融合 IQ 基带数据与气象/高度/方位等多维传感器信息
- **边缘 AI 推理**：利用 RK3588 NPU 实现毫秒级推理，无需依赖云端
- **闭环迭代架构**：板端采集 → PC 训练 → 板端部署，实现模型持续优化
