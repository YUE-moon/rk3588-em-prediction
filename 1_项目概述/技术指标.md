# 技术指标

## 硬件平台

| 参数 | 规格 |
|------|------|
| 主控芯片 | Rockchip RK3588 |
| 开发板 | ELF2 |
| CPU | 4×Cortex-A76(2.4GHz) + 4×Cortex-A55(1.8GHz) |
| NPU | 6 TOPS (INT8) |
| 内存 | 8GB LPDDR4 |
| 存储 | 64GB eMMC + microSD |

## 系统性能指标

| 指标 | 参数 |
|------|------|
| 推理延迟 | < 10ms（单次预测） |
| 预测精度 | RMSE < 3dB（路径损耗预测） |
| 数据采集速率 | 支持 IQ 数据实时采集 |
| 传感器通道 | 气象、高度、方位等多通道同步 |
| 工作频段 | 可配置（取决于信标与频谱仪） |

## 软件环境

| 组件 | 版本/说明 |
|------|-----------|
| 操作系统 | Linux（ARM64） |
| Python | 3.8+ |
| RKNN-Toolkit2 | PC 端模型转换工具 |
| RKNPU2 | 板端 NPU 运行时库 |
| 深度学习框架 | PyTorch / TensorFlow（PC 端训练） |

## 模型参数

| 参数 | 说明 |
|------|------|
| 输入特征 | IQ 数据特征 + 气象参数 + 高度 + 方位 |
| 输出 | 路径损耗预测值（dB） |
| 模型类型 | 神经网络回归模型 |
| 部署格式 | RKNN（INT8 量化） |
| 模型大小 | < 5MB |
